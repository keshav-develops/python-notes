````markdown
# Exercises – Comprehensions

## 1. Concept Questions
1. What is a list comprehension, and how does it differ from using a `for` loop?  
2. Explain the difference between a **list comprehension** and a **generator expression**.  
3. How does a **set comprehension** handle duplicate values?  
4. When should you prefer a **dict comprehension** over building a dictionary with a loop?  

---

## 2. Coding Exercises

### Q1. List Comprehension – Squares of Even Numbers
Write a list comprehension that generates the squares of all even numbers between 1 and 20.

---

### Q2. Dict Comprehension – Word Lengths
Given the list:
```python
words = ["python", "comprehension", "exercise"]
````

Create a dictionary where each word is the key and its length is the value.

---

### Q3. Set Comprehension – Unique Vowels

From the string `"comprehensions are powerful"`, extract a set of unique vowels.

---

### Q4. Generator Expression – Cubes

Write a generator expression that yields cubes of numbers from 1 to 10, and print them using a `for` loop.

---

## 3. Stretch Tasks

### Task 1 – Prime Numbers with Comprehensions

Write a list comprehension that generates all prime numbers between 1 and 100.

---

### Task 2 – Nested Comprehensions

Use a nested list comprehension to create a multiplication table (1–5).
The output should look like:

```
[[1, 2, 3, 4, 5],
 [2, 4, 6, 8, 10],
 [3, 6, 9, 12, 15],
 [4, 8, 12, 16, 20],
 [5, 10, 15, 20, 25]]
```
